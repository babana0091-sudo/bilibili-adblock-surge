# 安装

## 自动更新（推荐）

```text
https://raw.githubusercontent.com/babana0091-sudo/bilibili-adblock-surge/main/bilibili-adblock.sgmodule
```

Surge → 模块 → 安装新模块 → 粘贴上述链接 → 启用 → 开启 MITM + MITM over HTTP/2。

日常点模块「更新」即可跟随 `main` 最新内容。

## 固定版本

使用 [Releases](https://github.com/babana0091-sudo/bilibili-adblock-surge/releases) 中的 `.sgmodule` 资产。  
Release URL 带版本号，**不会**自动升到新版本。
